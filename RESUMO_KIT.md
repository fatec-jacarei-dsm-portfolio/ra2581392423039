## 🚀 Como Usar em Novos Projetos

### Método 1: Copie a Pasta `docs/`
```bash
cp -r docs/ meu-projeto/
# Edite index.html com seu conteúdo
```

### Método 2: Use o Template
```bash
# 1. Crie nova pasta
mkdir meu-projeto
cd meu-projeto

# 2. Crie estrutura
mkdir css js assets

# 3. Copie os arquivos base
cp TEMPLATE_ROCKETSEAT.html index.html
cp docs/css/styles.css css/
cp docs/js/script.js js/

# 4. Customize!
```

### Método 3: Integre em Projeto Existente
```bash
# 1. Copie apenas o CSS
cp docs/css/styles.css seu-projeto/css/

# 2. Link no HTML
<link rel="stylesheet" href="css/styles.css">

# 3. Use as classes
<button class="btn btn-primary">Clique</button>
```

## 💡 Dicas Importantes

### Para Começar Rápido
- Use o template como base
- Copie `styles.css` inteiro (não modifique nome)
- Mantenha a estrutura HTML proposta
- Customize apenas o que precisa

### Para Customização Avançada
- Edite variáveis CSS em `:root`
- Adicione suas animações em `@keyframes`
- Estenda componentes conforme necessário
- Mantenha naming consistente

### Para Melhor Performance
- Comprima imagens (TinyPNG, Squoosh)
- Minifique CSS/JS para produção
- Use CDN para fontes (já incluído)
- Lazy load imagens pesadas

### Para SEO
- Estrutura HTML semântica ✅ (já feita)
- Meta tags apropriadas
- Schema.json (markup estruturado)
- Sitemap.xml
- Robots.txt
